Record Website

For my class we were each asked to come up with a website idea that we thought would be interesting. I'm a big fan of the new vinyl record trend so I decided to base it on why they are becoming popular again and how vinyl records work. Each week over the semester I learned new concepts to apply to the website until I had applied everything I had learned. This included using CSS for things such as hover states or affecting image sizes so they would be the same. The link below will take you to the finalized record website I created using HTML.
